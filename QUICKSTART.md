# 🚀 Quick Start Guide

Die schnellste Methode, um MagicMirror Setup auf deinem Raspberry Pi zu installieren.

## ⚡ Ein-Befehl Installation

```bash
curl -fsSL https://raw.githubusercontent.com/twicemind/magicmirror-setup/main/install.sh | sudo bash
```

Das war's! 🎉

## 📋 Was wird installiert?

- ✅ Automatische System-Updates
- ✅ Docker-Container-Verwaltung
- ✅ Modul-Update-System
- ✅ WebUI auf Port 8080
- ✅ Alle Management-Scripts

## 🖥️ WebUI öffnen

Nach der Installation:

```
http://<deine-raspberry-pi-ip>:8080
```

Beispiel:
```
http://192.168.1.100:8080
```

## ⚙️ Erste Schritte

### 1. Konfiguration anpassen

WebUI öffnen → Configuration Editor → config.json bearbeiten → Save → Restart Container

### 2. Modul hinzufügen

WebUI öffnen → Installed Modules → Git URL eingeben → Install Module

### 3. Display-Orientierung

WebUI öffnen → Display Settings → Orientierung wählen → Apply
